# 税務書類リスト (Tax Docs)

所得税・贈与税の申告に必要な書類を管理するための Web アプリケーションです。
開いた瞬間から編集・印刷できる1画面統合ビューで、ドラッグ&ドロップによる並び替え、Excel/JSON形式でのデータ入出力機能を提供します。

## 主な機能

### 申告種別
- **所得税申告 / 贈与税申告の切り替え**: ドロップダウンで申告種別を選択、書類リストが自動切替
- **対象年度選択**: 年度を変更すると書類テキスト内の令和年が自動更新

### 書類管理機能
- **必要書類の案内**: 申告種別に応じた書類を動的に表示
- **カテゴリ管理**: 書類をカテゴリ（大分類）ごとに整理
- **階層番号表示**: 大分類①②③・中分類1. 2. 3.・小分類1-1, 1-2
- **中項目対応**: 各書類に対して詳細な中項目（サブアイテム）を追加可能
- **特例マーク**: 特例/控除に関連するカテゴリを視覚的に区別（紫色）

### 編集機能
- **ドラッグ&ドロップ**: カテゴリ・書類の並び替えに対応（@dnd-kit使用）
- **インライン編集**: カテゴリ名・書類名・中項目をその場で編集（Enter確定/Escape取消）
- **提出済み管理**: チェックを入れると提出済み（取消線表示）、印刷時に非表示オプション
- **全済みボタン**: カテゴリ内の全書類を一括で提出済み/未提出に切替
- **一括展開/折りたたみ**: 全カテゴリの表示状態を一括操作

### データ入出力
- **Excel出力**: スタイル付きのExcelファイルとしてダウンロード
- **ファイルに保存**: 編集内容をJSONファイルとして保存
- **ファイルを読込**: 保存したJSONファイルを読み込んで編集を再開
- **印刷**: 1段組/2段組の切替、全表示/提出済み非表示の切替に対応

### 顧客・担当者情報
- **担当者名・連絡先**: localStorage に保存（ブラウザ閉じても保持）
- **顧客名**: sessionStorage に保存（タブを閉じるとクリア）
- 印刷時のヘッダー・フッター、ファイル名に反映

### UI/UX
- **ダークモード**: ワンクリックでライト/ダーク切替（localStorage保存）
- **レスポンシブ**: PC表示はインラインツールバー、モバイルはハンバーガーメニュー
- **外部リンク**: 国税庁チェックシート、e-Tax添付書類ページへの直接リンク

## Docker

```bash
# 開発環境
docker compose up -d

# 再ビルド
docker compose up -d --build

# 停止
docker compose down
```

アクセス: http://localhost:3002/tax-docs/

> `manage.bat start` で全アプリを起動する場合は http://localhost/tax-docs/ からアクセス可能
