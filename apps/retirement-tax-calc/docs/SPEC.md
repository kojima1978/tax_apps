# 退職金 税額計算シミュレーター 仕様書

## 1. 概要

退職金にかかる所得税・復興特別所得税・住民税を計算する業務用Webアプリケーション。税理士法人の担当者が、社内での税額計算および顧客への説明・提案に使用することを想定。全UIは日本語。

### 対象ユーザー

- 税理士法人のスタッフ（PC利用が主、顧客説明時にも使用）

### 動作環境

- モダンブラウザ（Chrome, Edge, Firefox, Safari）
- Docker コンテナ上で動作（ポート: 3013, basePath: `/retirement-tax-calc`）

## 2. 画面仕様

### 2.1 レイアウト

1画面完結型（ルーティングなし）。上下配置でフォーム（上）+ 結果（下）。

#### ヘッダー

| 要素 | 仕様 |
|------|------|
| Homeボタン | アイコン → `/`（ポータルトップ） |
| タイトル | 「退職金 税額計算シミュレーター」 |
| PDFボタン | PDFファイルをダウンロード |
| 印刷ボタン | ブラウザ印刷ダイアログ |

### 2.2 入力フォーム

#### 退職金支給額（3パターン横並び）

| パターン | 色 | CSS変数 |
|----------|------|---------|
| 案① | 緑 | `--pattern-0: #2e7d32` |
| 案② | 青 | `--pattern-1: #1565c0` |
| 案③ | オレンジ | `--pattern-2: #e65100` |

- 各パターンに金額（円）を入力
- 自動カンマ挿入、全角数字→半角変換
- フォーカス時に値を全選択（上書き入力しやすく）
- 1つ以上の金額入力 + 勤続年数 > 0 で計算可能

#### 役員退職金チェック

| 項目 | 仕様 |
|------|------|
| 表示条件 | 退職区分が「短期退職手当等」以外の場合 |
| チェックON | 役員限度額セクションを表示 |

#### 役員限度額セクション

| 項目 | 型 | 説明 |
|------|-----|------|
| 最終月額報酬 | number（円） | 退職時の月額報酬 |
| 役職プリセット | ボタン群 | 社長(3.0), 専務(2.5), 常務(2.5), 取締役(2.0), 監査役(2.0) |
| 功績倍率 | number | プリセットまたは手動入力 |
| 勤続年数 | number | フォーム共通の勤続年数と連動 |

- **限度額** = 最終月額報酬 × 功績倍率 × 勤続年数
- 算式と計算結果を表示
- 「案①に適用」「案②に適用」「案③に適用」ボタンで金額をセット
- 案①の支給額が限度額を超える場合は警告（超過額を表示）
- 限度額内の場合はOK表示

#### 条件設定（2列グリッド）

| 項目 | 型 | 選択肢・仕様 |
|------|-----|-------------|
| 退職区分 | ラジオ | 一般退職手当等 / 特定役員退職手当等（勤続5年以下） / 短期退職手当等（勤続5年以下） |
| 勤続年数 | number / date | 直接入力モード or 日付計算モード（トグルで切替） |
| 障害者退職 | checkbox | 「障害者になったことに直接基因して退職した」→ 控除額に100万円加算 |
| 適用税率年度 | select | 令和6年(2024) / 令和7年(2025) |

**勤続年数 — 日付計算モード:**
- 勤続開始日・退職日を入力 → 年数を自動計算（1年未満切上げ）
- 計算結果をノートで表示

#### ボタン行

| ボタン | 仕様 |
|--------|------|
| 計算する | 全パターン一括計算。disabled: 金額0かつ勤続年数0。ショートカット: `Ctrl+Enter` |
| クリア | 全入力・結果を一括リセット |

## 3. 計算ロジック

### 3.1 退職所得控除額

| 勤続年数 | 計算式 |
|----------|--------|
| 20年以下 | 40万円 × 勤続年数（最低80万円） |
| 20年超 | 800万円 + 70万円 ×（勤続年数 − 20年） |
| 障害者退職 | 上記 + 100万円 |

### 3.2 課税退職所得金額

| 退職区分 | 計算式 |
|----------|--------|
| 一般退職手当等 | （支給額 − 控除額）× 1/2 |
| 特定役員退職手当等 | 支給額 − 控除額（1/2なし） |
| 短期退職手当等 | 300万円以下: × 1/2、超過部分: 1/2なし |

- 1,000円未満切捨て

### 3.3 所得税額

- 速算表（7段階）により算出
- 100円未満切捨て

### 3.4 復興特別所得税額

- 所得税額 × 2.1%
- 1円未満切捨て

### 3.5 住民税額

- 市民税: 課税退職所得金額 × 6%（100円未満切捨て）
- 県民税: 課税退職所得金額 × 4%（100円未満切捨て）

### 3.6 役員退職金限度額

- 限度額 = 最終月額報酬 × 功績倍率 × 勤続年数

### 3.7 勤続年数計算（日付モード）

- 開始日〜退職日の年数差 + 端数日がある場合は切上げ
- 最低1年

### 3.8 実効税率

- 税額合計 ÷ 支給額 × 100（小数第2位まで）

## 4. PDF出力

- html2pdf.js を使用（html2canvas + jsPDF）
- A4縦、余白8mm
- ボタン類・警告バナー・プリセットボタン等は非表示（`.pdf-generating` クラス）
- ファイル名: `退職金税額計算.pdf`

## 5. 印刷

- `@media print` で A4 縦レイアウト
- **2ページ構成**: 1ページ目: ヘッダー＋入力フォーム、2ページ目: 計算結果セクション以降（`break-before: page`）
- ボタン類・変更通知・モードトグル・プリセットは非表示
- フォーム入力は透明背景＋下線表示（手書き可能）
- 未選択ラジオは非表示
- フッターに会社情報・担当者・作成日

## 6. レスポンシブ対応

| ブレークポイント | 対応 |
|------------------|------|
| > 700px | 2列グリッド、3パターン横並び |
| ≤ 700px | 1列スタック、パターン縦並び |
| ≤ 600px | ヘッダー圧縮、パディング縮小、テーブル横スクロール |
| ≤ 380px | 極小スマホ対応（フォント・パディング最小化） |

## 7. アクセシビリティ

- フォームラベル: `htmlFor` + `useId()` によるID紐付け
- ARIAラベル: Homeボタンに `aria-label="ポータルへ戻る"`
- セマンティックHTML: `<header>`, `<label>`, `<table>`, `<select>`
- キーボード操作: `Ctrl+Enter` で計算実行
- カラーコントラスト: WCAG AA準拠
- `prefers-reduced-motion: reduce` でアニメーション無効化
