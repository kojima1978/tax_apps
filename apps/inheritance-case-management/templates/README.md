# テンプレートフォルダ

見積書・請求書のExcelテンプレートをこのフォルダに配置してください。

## ファイル名

| ファイル名 | 用途 |
|---|---|
| `estimate_template.xlsx` | 見積書・請求書共通テンプレート |
| `invoice_request_template.xlsx` | 請求書発行依頼票テンプレート |

見積書・請求書は共通の1ファイルを使用します。請求書出力時はコード側でタイトル・金額ラベル・振込先等の7セル（B11/B14/B17/B41/B43/B44/E44）を上書きします。

請求書発行依頼票は案件データ（担当者・請求先・紹介者・売上/紹介料・立替金）をセルに転記します。

**テンプレートが存在しない場合はエラーになります。必ず配置してください。**

## 注意

- このフォルダ内のExcelファイルは `.gitignore` で除外されています
- Dockerコンテナ内では `/app/templates/` にマウントされます
