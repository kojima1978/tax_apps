# テンプレートフォルダ

見積書・請求書のExcelテンプレートをこのフォルダに配置してください。

## ファイル名

| ファイル名 | 用途 |
|---|---|
| `estimate_template.xlsx` | 見積書テンプレート |
| `invoice_template.xlsx` | 請求書テンプレート |

テンプレートが存在しない場合は、コード内で自動生成されたレイアウトが使用されます。

## 注意

- このフォルダ内のExcelファイルは `.gitignore` で除外されています
- Dockerコンテナ内では `/app/templates/` にマウントされます
