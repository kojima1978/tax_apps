# テンプレートフォルダ

見積書・請求書のExcelテンプレートをこのフォルダに配置してください。

## ファイル名

| ファイル名 | 用途 |
|---|---|
| `estimate_template.xlsx` | 見積書・請求書共通テンプレート |
| `invoice_request_template.xlsx` | 請求書発行依頼票テンプレート |

見積書・請求書は共通の1ファイルを使用します。請求書出力時はコード側でタイトル・金額ラベル・振込先等の7セル（B11/B14/B17/B41/B43/B44/E44）を上書きします。

請求書発行依頼票は案件データ（担当者・請求先+被相続人名・紹介者・手取り額/紹介料・報酬額税抜・立替金）をセルに転記します。

## 見積書・請求書テンプレートの転記セル

`estimate_template.xlsx` には、見積書・請求書共通で以下のセルへ転記します。

| 項目 | セル | 備考 |
|---|---:|---|
| 宛名 | D2 | 選択した宛名 |
| 被相続人名 | H15 | 案件の被相続人名 |
| 遺産総額 | H22 | 報酬計算の遺産総額 |
| 土地数（路線価） | K27 | 件数 |
| 土地数（倍率） | K28 | 件数 |
| 非上場株式の数 | K30 | 件数 |
| 相続人の数 | J32 | 人数 |
| 特別業務報酬額 1行目 内容 | B35 | 先頭に半角スペース4つを付与、MS 明朝 9pt 太字 |
| 特別業務報酬額 1行目 金額 | M35 | 金額 |
| 特別業務報酬額 2行目 内容 | B36 | 先頭に半角スペース4つを付与、MS 明朝 9pt 太字 |
| 特別業務報酬額 2行目 金額 | M36 | 金額 |
| 値引額 | M38 | マイナス値で転記（例: 1,000 -> -1,000）。M37 はテンプレート側の関数を保持するため上書きしない |
| 差引額（税抜） | M39 | 見積書は見積額、請求書は報酬額を転記 |
| 立替金合計 | M42 | 請求書のみ転記。見積書では上書きしない |

見積書は案件詳細の「見積書」欄の数字（見積額・見積紹介料）を基準に出力します。請求書と請求書発行依頼票は「請求書」欄の数字（報酬額・紹介料）を基準に出力します。立替金は請求書と請求書発行依頼票にのみ転記します。

請求書出力時は `B42` を ` ４．立替金費用` に置き換え、MS 明朝 10pt 太字にします。

**テンプレートが存在しない場合はエラーになります。必ず配置してください。**

## 注意

- このフォルダ内のExcelファイルは `.gitignore` で除外されています
- Dockerコンテナ内では `/app/templates/` にマウントされます
