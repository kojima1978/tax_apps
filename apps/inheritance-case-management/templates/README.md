# テンプレートフォルダ

見積書・請求書・請求書発行依頼票の Excel テンプレートをこのフォルダに配置します。

## ファイル名

| ファイル名 | 用途 |
|---|---|
| `estimate_template.xlsx` | 見積書・請求書共通テンプレート |
| `invoice_request_template.xlsx` | 請求書発行依頼票テンプレート |

見積書・請求書は共通の `estimate_template.xlsx` を使用します。請求書出力時はコード側でタイトル、金額ラベル、振込先などを上書きします。

請求書発行依頼票は、案件データの担当者、請求先と被相続人名、紹介者、手取り額、紹介料、報酬額税抜、立替金をセルに転記します。

## 見積書・請求書テンプレートの転記セル

`estimate_template.xlsx` には、見積書・請求書共通で以下のセルへ転記します。

| 項目 | セル | 備考 |
|---|---:|---|
| 宛名 | D2 | 選択した宛名 |
| 被相続人名 | H15 | 案件の被相続人名 |
| 遺産総額 | H22 | 報酬計算の遺産総額 |
| 土地数（路線価） | K27 | 件数 |
| 土地数（倍率） | K28 | 件数 |
| 非上場株式の数 | K30 | 件数 |
| 相続人の数 | J32 | 人数 |
| 特別業務報酬額 1行目 内容 | B35 | 先頭に半角スペース4つを付与、MS 明朝 9pt 太字 |
| 特別業務報酬額 1行目 金額 | M35 | 金額 |
| 特別業務報酬額 2行目 内容 | B36 | 先頭に半角スペース4つを付与、MS 明朝 9pt 太字 |
| 特別業務報酬額 2行目 金額 | M36 | 金額 |
| 値引額 | M38 | マイナス値で転記。例: 1,000 -> -1,000。M37 はテンプレート側の関数を保持するため上書きしない |
| 差引額（税抜） | M39 | テンプレート側の数式を保持するため上書きしない |
| 立替金合計 | M42 | 請求書のみ転記。見積書では上書きしない |

見積書は案件詳細の「見積書」欄の数字、つまり見積額・見積紹介料を基準に出力します。

請求書と請求書発行依頼票は、案件詳細の「請求書」欄の数字、つまり報酬額・紹介料を基準に出力します。

立替金は請求書と請求書発行依頼票にのみ転記します。見積書では立替金セルを上書きしません。

請求書出力時は `B42` を ` ４．立替金費用（戸籍謄本・不動産登記事項閲覧・残高証明書発行手数料等）` に置き換え、MS 明朝 10pt 太字にします。

## 請求書出力時の上書き

請求書出力時は `estimate_template.xlsx` に対して、以下のセルも請求書用に上書きします。

| セル | 内容 |
|---:|---|
| B11 | 請求書タイトル |
| B14 | 請求文 |
| B17 | 金額ラベル |
| B42 | ` ４．立替金費用（戸籍謄本・不動産登記事項閲覧・残高証明書発行手数料等）` |
| B43 | 御請求額ラベル |
| B44 | 振込先ラベル |
| E44 | 振込先情報 |

## 注意

- テンプレートが存在しない場合はエラーになります。必ず配置してください。
- このフォルダ内の Excel ファイルは `.gitignore` で除外されています。
- Docker コンテナ内では `/app/templates/` に読み取り専用でマウントされます。
