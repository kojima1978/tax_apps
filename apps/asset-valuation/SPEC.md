# Asset Valuation（相続税 減価償却資産評価）機能仕様書

## 1. 概要

相続税申告における減価償却資産の評価額を算出するWebアプリケーション。
会計ソフトから出力されたCSVファイルを取り込み、相続税法の評価通達に基づいて計算し、Excel出力・印刷を行う。

### 1.1 技術スタック

| 項目 | 技術 |
|------|------|
| フロントエンド | Vite 7.x + React 19.x + TypeScript 5.x |
| スタイリング | Tailwind CSS v4（@tailwindcss/postcss） |
| アイコン | lucide-react |
| Excel出力 | xlsx-js-style |
| テーマカラー | 緑（#2e7d32） |
| Docker ポート | 3017 |
| basePath | `/asset-valuation` |

### 1.2 アーキテクチャ

- **完全フロントエンド**（バックエンド不要）
- 未償却残高表はJSONデータとして内蔵（H24.4.1以後取得分）
- CSV解析はブラウザ内で完結（最大200件）
- データ永続化: JSON入出力（案件データ・マッピングプリセット）
- Docker: Vite dev → nginx:1.27-alpine 静的配信

## 2. 画面仕様

### 2.1 画面構成

単一ページ・ステップ形式（4ステップ）

```
Step 1: CSVインポート + 基本情報入力
    ↓
Step 2: カラムマッピング
    ↓
Step 3: データ確認・編集
    ↓
Step 4: 計算結果 → Excel出力 / 印刷
```

ステップ間は「次へ」「戻る」ボタンで遷移。各ステップにバリデーションあり。

---

### 2.2 Step 1: CSVインポート + 基本情報入力

| 項目 | 仕様 |
|------|------|
| 案件名 | テキスト入力（必須）。例：「株式会社〇〇〇 様」 |
| 課税時期 | 日付入力（必須） |
| CSVファイル | ファイルドロップ or ファイル選択 |
| エンコーディング | UTF-8, CP932 (Shift_JIS) 自動検出 |
| 案件JSONインポート | JSONファイルをアップロードして過去案件を復元 |

#### CSV要件
- ヘッダー行あり（1行目）
- 最大300件の資産データ

## 3. 計算仕様

### 3.1 共通計算

#### 3.1.1 経過年数

```
経過年数 = ROUNDUP((課税時期 - 取得年月) / 365, 0)
```

- 1年未満の端数は1年に切り上げ

#### 3.1.2 3年以内判定

```
3年以内 = 取得年月 > (課税時期 - 365 × 3)
```

- 該当カテゴリ: 建物、建物付属設備、構築物

### 3.2 カテゴリ別計算ルール

#### 3.2.1 建物

| 条件 | 評価根拠 | 計算式 |
|------|---------|--------|
| 固定資産税評価明細あり | 評基通89 | 相続税評価額 = `-`（対象外） |
| 3年以内取得 | 3年内簿価 | 相続税評価額 = 期末簿価 |
| 上記以外 | 評基通89-2(2) | 相続税評価額 = `(取得価額 - 償却額) × 0.7` |

**賃貸の場合（建物）:** 上記で算出した相続税評価額に対し、さらに `× 0.7`（借家権割合30%控除）を適用。
固定資産税評価明細の行（評価額 = `-`）には適用しない。

**建物の償却額（定額法）:**

```
経過年数 ≧ 耐用年数の場合:
  償却額 = 取得価額 × 0.9

経過年数 < 耐用年数の場合:
  償却額 = 取得価額 × 0.9 × (経過年数 / 耐用年数)
```

**判定優先順位:** 固定資産税評価明細 > 3年以内 > 89-2(2)

#### 3.2.2 建物付属設備

| 条件 | 評価根拠 | 計算式 |
|------|---------|--------|
| 固定資産税評価明細あり | 評基通89 | 相続税評価額 = `-`（対象外） |
| 3年以内取得 | 3年内簿価 | 相続税評価額 = 期末簿価 |
| 上記以外 | 評基通92 | 相続税評価額 = `ROUNDDOWN(取得価額 × 未償却残額割合 × 0.7, 0)` |

**賃貸の場合（建物付属設備）:** 上記で算出した相続税評価額に対し、さらに `× 0.7`（借家権割合30%控除）を適用。
固定資産税評価明細の行（評価額 = `-`）には適用しない。

#### 3.2.3 構築物

| 条件 | 評価根拠 | 計算式 |
|------|---------|--------|
| 3年以内取得 | 3年内簿価 | 相続税評価額 = 期末簿価 |
| 上記以外 | 評基通97 | 相続税評価額 = `ROUNDDOWN(取得価額 × 未償却残額割合 × 0.7, 0)` |

#### 3.2.4 機械装置

| 評価根拠 | 計算式 |
|---------|--------|
| 評基通129 | 相続税評価額 = `ROUNDDOWN(取得価額 × 未償却残額割合, 0)` |

#### 3.2.5 車両及び運搬具

| 評価根拠 | 計算式 |
|---------|--------|
| 評基通129 | 相続税評価額 = `ROUNDDOWN(取得価額 × 未償却残額割合, 0)` |

#### 3.2.6 器具及び備品

| 評価根拠 | 計算式 |
|---------|--------|
| 評基通129 | 相続税評価額 = `ROUNDDOWN(取得価額 × 未償却残額割合, 0)` |

### 3.3 未償却残額割合

**定率法未償却残額表（平成24年4月1日以後取得分）** を使用。

```
未償却残額割合 = rateTable[経過年数][耐用年数]
```

- 経過年数 ≧ 耐用年数の場合: 未償却残額割合 = 0
- テーブル範囲: 経過年数 1〜50、耐用年数 2〜50

## 4. Excel出力仕様

### 4.1 レイアウト

sample.xlsmの「減価償却資産」シートを再現する。

```
行1: 案件名（「株式会社〇〇〇　様」）
行2: 課税時期: YYYY/MM/DD
行3: 3年以内: YYYY/MM/DD
行4: （空行）
行5: 【カテゴリ名】
行6: ヘッダー行（NO, 名称, 取得年月, 課税時期, 経過年数, 耐用年数, 取得価額, 償却額/償却率, 相続税評価額, 期末簿価, その他）
行7〜: データ行
行N: 合計行（取得価額合計, 相続税評価額合計, 期末簿価合計）
（空行）
次のカテゴリ...
（全カテゴリ終了後）
評価通達条文テキスト
```

### 4.2 カラム仕様

| カラム | 建物 | 建物付属設備/構築物/機械装置/車両/器具備品 |
|--------|------|----------------------------------------|
| H列 | 償却額（金額） | 償却率（小数） |

### 4.3 書式

| 要素 | 書式 |
|------|------|
| フォント | MS ゴシック or 游ゴシック |
| ヘッダー行 | 背景色付き、太字 |
| 金額 | 3桁カンマ区切り、右寄せ |
| 日付 | YYYY/MM/DD |
| 合計行 | 太字、上罫線 |
| 評価額 `-` | 中央寄せ |

### 4.4 評価通達条文

Excel末尾に以下の条文テキストを記載:

- 評基通89-2（文化財建造物である家屋の評価）
- 評基通92（附属設備等の評価）
- 評基通97（構築物の評価）
- 評基通128〜130（一般動産の評価・償却費の額の計算）

## 6. データ仕様

### 6.1 未償却残高表

**定率法未償却残額表（平成24年4月1日以後取得分）** をJSONとしてアプリに内蔵。

- 行: 経過年数（1〜50）
- 列: 耐用年数（2〜50）
- 値: 未償却残額割合（小数、0〜1）
- データソース: sample.xlsmの「未償却残高表」シート

### 6.2 資産カテゴリ

| カテゴリ | 償却方法 | ×0.7 | 固定資産税評価明細 | 3年以内判定 | 賃貸控除 |
|---------|---------|------|-----------------|-----------|---------|
| 建物 | 定額法 | あり | あり | あり | あり |
| 建物付属設備 | 定率法 | あり | あり | あり | あり |
| 構築物 | 定率法 | あり | なし | あり | なし |
| 機械装置 | 定率法 | なし | なし | なし | なし |
| 車両 | 定率法 | なし | なし | なし | なし |
| 器具備品 | 定率法 | なし | なし | なし | なし |

## 7. バリデーション

| チェック項目 | タイミング | 種別 |
|------------|----------|------|
| 案件名が空 | Step 1 | エラー |
| 課税時期が未入力 | Step 1 | エラー |
| 必須カラムが未マッピング | Step 2 | エラー |
| 耐用年数が範囲外（2〜50） | Step 3 | エラー |
| 取得年月が課税時期より未来 | Step 3 | 警告 |
| 取得価額が0以下 | Step 3 | エラー |
| 経過年数が耐用年数を大幅超過（2倍以上）かつ簿価が大きい | Step 3 | 警告 |
