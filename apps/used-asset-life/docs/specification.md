# 中古資産の耐用年数・簿価計算 — 仕様書

## 1. 概要

法人が中古資産を取得した場合に、耐用年数の計算・減価償却スケジュールの作成・任意期間の償却額算出を行うWebツール。
国税庁タックスアンサー No.5404「中古資産の耐用年数」および減価償却関連通達に準拠。

3つのタブで構成され、タブ間の計算結果の連携が可能。

## 2. タブ構成

| タブ | ラベル | デフォルト | 概要 |
|------|--------|-----------|------|
| five_year | 期間償却 | ○ | 任意期間の償却スケジュール表示 |
| life | （中古）耐用年数計算 | | 簡便法による耐用年数算出 |
| depreciation | 簿価計算 | | 全期間の償却スケジュール + 基準日簿価 |

### タブ間連携フロー

```
耐用年数計算 → 簿価計算 → 期間償却
```

- 耐用年数計算の結果（耐用年数・取得価額・取得日）を簿価計算に引き継ぎ
- 簿価計算の条件（全入力値）を期間償却に引き継ぎ
- 各タブの入力状態はタブ切替時も保持

## 3. タブ1: 期間償却

### 3.1 入力項目

| 項目 | 型 | 必須 | 説明 |
|------|------|------|------|
| 資産の種類 | select | ○ | 建物, 建物附属設備, 構築物, 機械装置, 車両運搬具, 器具備品, その他 |
| 取得価額 | currency | ○ | カンマ区切り入力 |
| 耐用年数 | number | ○ | 2〜100年。資産種類に応じたプリセットボタン付き |
| 取得日 | date | ○ | 償却方法の推奨判定に使用（H19.4 / H24.4 境界） |
| 事業供用日 | date | ○ | 月割計算の起算日。取得日からの複写ボタン付き |
| 償却方法 | select | ○ | 定額法, 定率法, 旧定額法, 旧定率法（全方法選択可、推奨表示あり） |
| 決算月 | select | ○ | 1〜12月（デフォルト: 3月） |
| 開始年度 | number | - | 表示する開始年度（西暦）。未入力時は初年度から |
| 表示年数 | number | ○ | 表示する年数。プリセット: 3, 5, 10, 20年（デフォルト: 5年） |

### 3.2 出力項目

| 項目 | 説明 |
|------|------|
| 条件タグ | 償却方法, 耐用年数, 取得価額, 償却率 |
| サマリーカード（3枚） | N年間合計償却額（緑ハイライト）, 開始時簿価, N年後簿価 |
| 償却スケジュール表 | 期, 会計期間, 期首帳簿価額, 償却額, 期末帳簿価額, 備考 |
| 免責事項 | 概算である旨の注記 |

### 3.3 計算ロジック

1. `calcDepreciationSchedule()` で全期間の償却スケジュールを算出
2. 開始年度に基づき、periodLabel の開始年 ≥ startYear となる行を検索
3. 表示年数分をスライスして表示
4. スライス内の合計償却額・開始/終了簿価を算出

## 4. タブ2: （中古）耐用年数計算

### 4.1 計算ルール

#### 簡便法（法定耐用年数を超えていない場合）

```
中古耐用年数 = (法定耐用年数 - 経過年数) + 経過年数 × 20%
```

- 内部計算は月単位で行い、最後に年に変換する
- 1年未満の端数は切捨て
- 計算結果が2年未満の場合は2年とする

#### 簡便法（法定耐用年数を全部経過している場合）

```
中古耐用年数 = 法定耐用年数 × 20%
```

- 同様に月単位計算 → 年変換 → 端数切捨て → 最低2年

#### 50%ルール

改修・資本的支出額が取得価額の50%を超える場合、簡便法は適用できず、法定耐用年数をそのまま使用する。

### 4.2 入力項目

| 項目 | 型 | 必須 | 説明 |
|------|------|------|------|
| 資産の種類 | select | ○ | 建物, 建物附属設備, 構築物, 機械装置, 車両運搬具, 器具備品, その他 |
| 法定耐用年数 | number | ○ | 1〜100年。資産種類に応じたプリセットボタン付き |
| 新築・製造日 | date | - | 経過年数の自動計算に使用 |
| 取得日 | date | - | 経過年数の自動計算に使用 |
| 経過年数 | number | ○ | 年（0〜100）+ 月（0〜11）。日付入力時は自動計算 |
| 取得価額 | currency | - | カンマ区切り入力。50%ルール判定に使用 |
| 改修・資本的支出額 | currency | - | カンマ区切り入力。50%ルール判定に使用 |

### 4.3 出力項目

| 項目 | 説明 |
|------|------|
| 条件タグ | 資産種類, 法定耐用年数, 経過年数, 使用算式 |
| 50%判定結果 | 簡便法適用可/不可のバッジ + 詳細説明 |
| 計算過程 | 使用算式の表示 + ステップごとの計算過程 |
| 中古耐用年数 | 最終結果（大きく緑色で表示） |
| 連携ボタン | 「この結果で簿価計算へ →」 |
| 免責事項 | 概算である旨の注記 |

### 4.4 経過年数の自動計算

- 新築・製造日と取得日の両方が入力されると、経過年数（年・月）を自動計算
- 自動計算中は経過年数フィールドがreadOnlyになり、緑色のハイライト表示
- 経過年数を直接手入力すると自動計算モードが解除される
- 日付をクリアしても自動計算モードが解除される

## 5. タブ3: 簿価計算

### 5.1 入力項目

| 項目 | 型 | 必須 | 説明 |
|------|------|------|------|
| 資産の種類 | select | ○ | 建物, 建物附属設備, 構築物, 機械装置, 車両運搬具, 器具備品, その他 |
| 取得価額 | currency | ○ | カンマ区切り入力 |
| 耐用年数 | number | ○ | 2〜100年。資産種類に応じたプリセットボタン付き |
| 取得日 | date | ○ | 償却方法の推奨判定に使用 |
| 事業供用日 | date | ○ | 月割計算の起算日。取得日からの複写ボタン付き |
| 償却方法 | select | ○ | 全4方法選択可、推奨表示あり |
| 決算月 | select | ○ | 1〜12月（デフォルト: 3月） |
| 簿価算出基準日 | date | - | 指定時点の残存簿価をハイライト表示 |

### 5.2 償却方法の選択

- 取得日に関わらず全4方法（定額法, 定率法, 旧定額法, 旧定率法）を常に選択可能
- 取得日に基づき推奨方法に「（推奨）」ラベルを表示
- 非推奨方法を選択した場合、オレンジ色のアラートを表示（計算はブロックしない）
- 取得日変更時に推奨方法へ自動切替（現在の方法が非推奨の場合のみ）

#### 償却方法の推奨判定

| 取得日 | 推奨方法 |
|--------|---------|
| H19.3.31 以前 | 旧定額法, 旧定率法 |
| H19.4.1 〜 H24.3.31 | 定額法, 定率法（250%） |
| H24.4.1 以降 | 定額法, 定率法（200%） |

### 5.3 出力項目

| 項目 | 説明 |
|------|------|
| 条件タグ | 償却方法, 耐用年数, 取得価額, 決算月 |
| 適用償却率 | 償却率, 改定償却率（定率法のみ）, 償却保証額（定率法のみ） |
| 基準日簿価 | 指定時点の残存簿価（大きく緑枠で表示） |
| 償却スケジュール表 | 期, 会計期間, 期首帳簿価額, 償却額, 期末帳簿価額, 備考 |
| 連携ボタン | 「この条件で期間償却へ →」 |
| 免責事項 | 概算である旨の注記 |

### 5.4 テーブルの色分け

| 背景色 | 意味 |
|--------|------|
| 青（bg-blue-50） | 改定償却率への切替年度 |
| オレンジ（bg-orange-50） | 95%限度額到達年度 |
| 緑（bg-green-50） | 備忘価額1円到達年度 |

## 6. 共通仕様

### 6.1 操作

| 操作 | 説明 |
|------|------|
| 計算する | 入力値から計算実行。結果セクションへスムーズスクロール |
| クリア | 全入力値・結果をリセット |
| Ctrl+Enter | 計算するボタンのショートカット（全タブ対応） |
| 印刷 | ブラウザの印刷ダイアログを表示 |
| ホームアイコン | ポータルトップへ遷移 |

### 6.2 バリデーション

- 必須項目が入力されると計算ボタンが有効化
- 計算後に入力値を変更すると「再計算してください」の警告表示（オレンジ帯）
- 非推奨償却方法の選択時はアラート表示（計算はブロックしない）

### 6.3 テーブルUX

- **スティッキーヘッダー**: テーブルのヘッダー行がスクロール時に固定表示
- **行ツールチップ**: 色付き行にtitle属性で備考内容を表示（モバイルで長押し確認可能）
- **横スクロールヒント**: モバイルでテーブルが横にはみ出す場合、右端にグラデーションシャドウを表示
- **モバイル凡例**: 備考列非表示時に色分けの凡例を表示

### 6.4 印刷最適化

- **印刷用ヘッダー**: アクティブタブ名を含むタイトルを印刷時に表示
- **備考列の全表示**: モバイルで非表示の備考列を印刷時は表示
- **ページ分割制御**: セクション・サマリーカード・テーブル行の適切なpage-break設定
- **A4縦**: 用紙サイズA4、余白10mm×8mm
- **非表示要素**: タブバー、計算/クリアボタン、連携ボタン、横スクロールヒント
- **印刷フッター**: 会社情報・担当者名（入力可）・作成日

### 6.5 参照リンク

ページ下部に常時表示:
- 国税庁 中古資産の耐用年数関連（No.5404, Q&A）
- 主な耐用年数表（建物, 構築物, 車両運搬具, 器具備品, 機械装置）
- 減価償却関連（No.2100, No.2105, No.2106, 償却率等表PDF）
