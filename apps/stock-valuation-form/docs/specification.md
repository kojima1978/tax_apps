# 取引相場のない株式の評価明細書 — 仕様書

## 1. 概要

| 項目 | 内容 |
|---|---|
| アプリ名 | stock-valuation-form |
| 正式名称 | 取引相場のない株式（出資）の評価明細書 |
| 目的 | 国税庁の評価明細書（第１表〜第８表）をブラウザ上で入力・自動計算・印刷するWebフォーム |
| ステータス | 開発中（Gateway未登録、prod スキップ対象） |
| 対象ユーザー | 税理士・税務担当者 |
| 言語 | 日本語 |

### 5.2 第１表の１ — 株主判定・会社規模

#### 入力項目

**会社情報**

| フィールド | キー | 型 |
|---|---|---|
| 会社名 | `companyName` | テキスト |
| 代表者 | `representative` | テキスト |
| 課税時期 | `taxDate` | テキスト |
| 事業年度（開始） | `fiscalStart` | テキスト |
| 事業年度（終了） | `fiscalEnd` | テキスト |
| 所在地 | `address` | テキスト |
| 業種（最大4つ） | `businessDesc_0`〜`businessDesc_3` | テキスト |
| 業種コード | `businessCode_0`〜`businessCode_3` | テキスト |
| 売上割合 | `salesRatio_0`〜`salesRatio_3` | 数値 |

**株主一覧（最大20行）**

| フィールド | キー | 型 |
|---|---|---|
| 氏名 | `sh_name_{i}` | テキスト |
| 同族関係者 | `sh_dozoku_{i}` | チェック（0/1） |
| 筆頭株主グループ | `sh_hittou_{i}` | チェック（0/1） |
| 続柄 | `sh_rel_{i}` | テキスト |
| 株式数 | `sh_shares_{i}` | 数値 |
| 議決権数 | `sh_votes_{i}` | 数値 |

#### 計算ロジック

```
議決権割合（同族） = 同族チェック株主の議決権合計 ÷ 全議決権合計 × 100
議決権割合（筆頭） = 筆頭チェック株主の議決権合計 ÷ 全議決権合計 × 100
```

**同族判定マトリクス**

| 条件 | 判定結果 |
|---|---|
| 同族割合 > 50% | 同族株主等 |
| 同族割合 ≤ 50% | 同族株主等以外 |

**少数株主の評価方式判定**（決定木）

1. 役員である → 原則的評価方式等
2. 役員でない かつ 中心的同族株主がいる → 原則的評価方式等
3. 中心的同族株主がいない かつ 中心的株主がいる → 配当還元方式
4. いずれにも該当しない → 原則的評価方式等

#### ユーザー操作

- 株主行の追加・削除（1〜20行）
- 議決権順でソート
- 空行の一括削除
- 会社情報のリセット

### 5.3 第１表の２ — 会社規模判定

#### 入力項目

| フィールド | キー | 型 |
|---|---|---|
| 常勤従業員数 | `regular_emp` | 数値 |
| パート等従業員数 | `part_emp` | 数値 |
| 業種区分 | `industry_type` | 選択（卸売、小売・サービス、その他） |
| 総資産価額 | `total_assets` | 数値 |
| 取引金額 | `transaction_amount` | 数値 |

#### 計算ロジック

```
従業員合計 = 常勤従業員 + floor(パート従業員 × 0.5)
```

**判定基準**

| 判定 | 条件 |
|---|---|
| 大会社（強制） | 従業員 ≥ 70人 |
| 大〜小 | 総資産・従業員・取引金額の3基準のうち最も有利な区分 |

**会社規模の結果**

| 区分 | Lの割合 |
|---|---|
| 大会社 | — |
| 中会社の大 | 0.90 |
| 中会社の中 | 0.75 |
| 中会社の小 | 0.60 |
| 小会社 | — |

### 5.4 第２表 — 特定の評価会社判定

- 類似業種比準要素の充足判定（要素1〜5）
- 特定の評価会社への該当判定パネル

### 5.5 第３表 — 一般の評価会社の株式価額

- 3つの評価方式の比較:
  - ① 類似業種比準価額（第４表から連動）
  - ② 1株当たり純資産価額（第５表から連動）
  - ③ 純資産価額の80%（該当する場合）
- セクション1: 原則的評価方式
- セクション2: 特例的評価方式

### 5.6 第４表 — 類似業種比準価額（3セクション構成）

**セクション1: 基本情報**
- 資本金額、発行済株式数、1株当たり資本金
- 50円換算株式数

**セクション2: 配当・利益計算**
- 直前期末以前3期分の配当金額
- 利益金額の計算
- 非経常的利益の控除

**セクション3: 比準計算**
- 類似業種の配当・利益・簿価純資産
- 比準割合の算出
- 最終的な比準価額の確定

### 5.7 第５表 — 1株当たり純資産価額

#### 入力項目

**資産・負債明細（最大20行）**

| フィールド | キー | 型 |
|---|---|---|
| 科目名 | `a_name_{i}` / `l_name_{i}` | テキスト |
| 備考 | `a_note_{i}` / `l_note_{i}` | テキスト（「株式等」「土地等」等） |
| 相続税評価額 | `a_eval_{i}` / `l_eval_{i}` | 数値 |
| 帳簿価額 | `a_book_{i}` / `l_book_{i}` | 数値 |

#### 計算ロジック

```
⑤ 相続税評価額純資産 = max(0, 資産評価額合計 − 負債評価額合計)
⑥ 帳簿価額純資産     = max(0, 資産帳簿合計 + 現物出資評価 − 現物出資帳簿 − 負債帳簿合計)
⑦ 評価差額           = max(0, ⑤ − ⑥)
⑧ 法人税等相当額     = floor(⑦ × 0.37)     ← 37%
⑨ 課税時期純資産     = ⑤ − ⑧
⑩ 発行済株式数       = 発行済株式 − 自己株式
⑪ 1株当たり純資産    = ⑩ > 0 ? floor(⑨ × 1000 ÷ ⑩) : —
⑫ 80%純資産価額      = floor(⑪ × 0.8)      ← 議決権割合 ≤ 50%の場合のみ
```

**計算過程の可視化**: CalcProcess パネルで計算フローを図示

### 5.8 第６表 — 特定の評価会社の株式価額

5種類の株式評価方式に対応:

| 種類 | 評価方法 |
|---|---|
| 比準要素数1の会社 | 類似業種比準25% + 純資産75%のブレンド |
| 株式保有特定会社 | 純資産価額方式 |
| 土地保有特定会社 | 純資産価額方式 |
| 開業後3年未満の会社 | 純資産価額方式 |
| 休業中・開業前の会社 | 純資産価額方式 |

### 5.9 第７表 — 株式保有特定会社の株式価額

- 配当受取割合の計算（直前2期分）
- 営業利益金額の計算
- 比準要素の修正（B・C要素）

### 5.10 第８表 — 株式保有特定会社（続）

- S1方式の純資産価額修正
- 株式評価額の調整
- 最終的な株式価額の算出
