# 動作確認用ダミーデータ

このフォルダの JSON は、画面上部の `読込(JSON)` からそのまま読み込めます。

読み込み前に別データが残っている場合は、必要に応じて `全データリセット` を実行してください。

## ファイル

- `01-general-large-company.json`
  - 一般の評価会社、大会社想定。
  - 第1表の1、第1表の2、第4表、第5表の基本連動確認用。
- `02-medium-company-with-dividend-rights.json`
  - 中会社想定。
  - 配当期待権、配当還元方式まわりの表示確認用。
- `03-stock-holding-specific-company.json`
  - 株式等保有特定会社想定。
  - 第7表、第8表、S1/S2 関連の確認用。
- `04-edge-zero-and-negative-values.json`
  - ゼロ値、マイナス値、未入力時判定の確認用。
  - 比準要素が 0 の強調表示や、⑪法人税の課税所得金額のマイナス入力確認に使います。
